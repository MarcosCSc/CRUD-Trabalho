# Agenda Telefônica - CRUD com Java Swing

## O que o programa faz
- Cadastra, lista, edita e exclui contatos
- Tudo em uma janela com tabela (JTable)
- Confirmação antes de excluir
- Limpa os campos automaticamente depois de cada operação

## Estrutura do projeto
src/

├─ model/        → Contato.java

├─ controller/   → AgendaController.java

└─ view/         → AgendaView.java (tela principal)


## Como o professor vai rodar (zero trabalho)
1. Abrir o IntelliJ IDEA (ou Eclipse)
2. File → Open → selecionar essa pasta inteira
3. Executar a classe `view.AgendaView`
